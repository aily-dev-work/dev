# アクセスを増やすためにやること（mens-body.com）

計測とインデックスは問題ありません。いま足りないのは **検索結果からの流入** です。  
新規ドメインは「すぐ人が来る」状態ではなく、次の2系統で伸ばします。

## 事実（いま）

- GA4 / Search Console / インデックス：OK
- 訪問者：ほぼ自分だけ → 技術バグではなく **露出不足**
- 競合キーワード（メンズ脱毛・日焼け止め等）は強いサイトが多い → 一夜では勝てない

## 今週（あなたがやる・効果が出やすい順）

### 1. Search Consoleで主要URLを再リクエスト（30分）

URL検査 → 各URLを開き「インデックス登録をリクエスト」：

9. `https://mens-body.com/sitemap.xml`（サイトマップ画面で送信済みか確認）

### 2. 無料の外部露出（週3本で十分）

記事URLをそのまま貼るだけでOKです（広告色が強すぎないタイトルにする）。

| 場所 | 例 |
|------|-----|
| X（Twitter） | 「メンズ脱毛、料金比較で迷ったらまずここ」＋記事URL |
| note | 失敗談＋チェックリスト記事を要約→本編へリンク |
| はてブ / 個人ブログ | 「夏前にやった清潔感対策」 |

**禁止に近いこと**：フォーラムへのコピペ連投、相互リンク業者、購入被リンク。Googleに不利です。

### 3. 有料（早く数字を見たい場合）

Google広告で「メンズ脱毛 おすすめ」「男性 日焼け止め」など **購入・比較意図** だけ日予算1,000〜3,000円からテスト。  
オーガニックが育つまでのブリッジになります。

## サイト側で実施した改善（今回のpush）

- トップのタイトル・H1・説明文を検索語寄りに変更
- 「よくある検索から読む」セクションを追加（主要収益ページへの内部リンク）
- GAの二重読み込みを解消（計測の正確性）

## 今月のコンテンツ方針（効きやすい）

長くて弱い「まとめ」より、次のような **具体クエリ1本=1記事** を優先：

- メンズ脱毛 料金 ○○円
- ヒゲ脱毛 何回
- メンズ日焼け止め 白浮きしない
- ジム週2 メニュー 胸

既存記事の更新日を動かしつつ、比較表・FAQを厚くする方が新規ドメインには有利です。

## 目標の目安

| 期間 | 目安 |
|------|------|
| 2〜4週 | Search Consoleに検索クエリが少し出る |
| 1〜3ヶ月 | ロングテールから安定流入 |
| それまで | 外部投稿 or 広告で種火を作る |

オーガニックだけで明日から人が増える保証はありません。  
「計測OK → インデックスOK → 露出を増やす」が次のフェーズです。
